# Ofícina de Robos

<img width="274" height="144" alt="f66e2c2f8887b403958d4ec34482af01" src="https://github.com/user-attachments/assets/f32164ef-a773-403d-8e2f-b34b51591293" />

Minijogo feito em **React Native** onde você monta seus robos parte por parte, batalha com oponentes e ganha moedas para atualizar e melhorar seu arsenal para uma tarefa da minha faculdade de engenharia da computação. 

- Colete partes de robo, cada uma com valores unicos de **vida** e **peso**, além de **habilidades** como lança chamas, lâminas ou regeneração de vida. Troque, concerte e combine partes únicas para montar robos customizados.

- Entre em batalhas estratégicas de turno, lembrando que as partes do seu robo influenciam na sua **velocidade** e **energia**, então escolha com sabedoria.

- **Loja**: Use suas moedas para comprar novas partes.  
- **Ofícina**: Arraste as partes para sua bancada para:
  - Construir novos robôs.
  - Reparar componentes quebrados.

---

## Requisitos
- Acesse via [Expo Snack](https://appetize.io/embed/b_vfyableb3rimkjc4gfht7aqrn4?device=pixel8&launchUrl=exp%3A%2F%2Fu.expo.dev%2F933fd9c0-1666-11e7-afca-d980795c5824%3Fruntime-version%3Dexposdk%253A52.0.0%26channel-name%3Dproduction%26snack%3D%2540marloushazu%252Ftt%26snack-channel%3DKOjiz5gSe9&params=%7B"EXDevMenuDisableAutoLaunch"%3Atrue%2C"EXKernelDisableNuxDefaultsKey"%3Atrue%7D&appearance=light&deviceColor=black&scale=auto&orientation=portrait&centered=both)
#### ou
- Node.js (LTS recomendado)
- Ambiente de desenvolvimento React Native (Android Studio ou Xcode para iOS)
- expo CLI ou react-native CLI.

### Para expo CLI
expo start

### Para React Native CLI:
npx react-native run-android
#### ou
npx react-native run-ios
